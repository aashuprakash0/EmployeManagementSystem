# Employee Management System (Java + MySQL + JDBC)

## Features
- Employee onboarding
- Role & department management
- Attendance tracking
- CRUD operations using JDBC
- Console-based UI

## Tech Stack
- Java
- MySQL
- JDBC

## Setup Instructions

1. Clone the repository
2. Run `sql/schema.sql` in MySQL
3. Update DB credentials in `DBConnection.java`
4. Run `MainApp.java`

## Sample Operations
- Add Employee
- View Employees
- Update Attendance
- Delete Employee
